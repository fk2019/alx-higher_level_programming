=======================
How to use print_sorted()
=======================
This file contains one instance method print_sorted() of MyList class
Example usage

        >>> MyList = __import__('1-my_list').MyList
        >>> my_list = MyList()
        >>> my_list.append(1)
        >>> my_list.append(5)
        >>> my_list.print_sorted()
        [1, 5]

        >>> my_list = MyList()
        >>> my_list.append(5)
        >>> my_list.append(3)
        >>> my_list.append(7)
        >>> my_list.print_sorted()
        [3, 5, 7]

        >>> my_list = MyList()
        >>> my_list.append(5)
        >>> my_list.append(-5)
        >>> my_list.append(2)
        >>> my_list.print_sorted()
        [-5, 2, 5]

        >>> my_list = MyList()
        >>> my_list.append(-1)
        >>> my_list.append(-8)
        >>> my_list.append(-3)
        >>> my_list.print_sorted()
        [-8, -3, -1]
