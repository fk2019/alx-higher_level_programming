===================
The ``test`` module
===================

Using ``factorial``
------------------

First import ``factorial`` from ``test``module:

    >>> from test import factorial

Now use it:
    >>> factorial(3)
    6

It also works with floats, but must be an exact integer
    >>> factorial(4.0)
    24

    >>> factorial(4.4)
    Traceback (most recent call last)
    ValueError: n must be an number

But not for strings
    >>> factorial("string")
    Traceback (most recent call last)
    TypeError: n must be a number

Or unusually large numbers
    >>> factorial(1e300)
    Traceback (most recent call last)
    OverflowError: n is too large

Or negative numbers
    >>> factorial(-8)
    Traceback (most recent call last)
    ValueError: n must be >= 0
